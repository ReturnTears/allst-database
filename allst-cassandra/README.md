## 下载和安装Cassandra
```text
1、登录https：//cassandra.apache.org/download/下载Cassandra的最新版

2、下载得到一个apache-cassandra-4.0.11-bin.tar.gz 压缩包

3、解压压缩包后文件夹为apache-cassandra-4.0.11，包含子目录如下：
   3.1、bin：该目录下包含了Cassandra的常用工具命令
   3.2、conf：保存Cassandra配置文件的目录
   3.3、doc：该目录的cql3子目录下有一份CQL.html文件，它是CQL的参考手册
   3.4、lib和pylib：这两个目录下保存了Cassandra程序的核心库及第三方依赖库
   3.5、tools：该目录下保存了Cassandra提供的相关工具
```
